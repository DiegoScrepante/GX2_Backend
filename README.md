# GX2_Backend

🚀 GX2 Backend

Backend desenvolvido para gerenciamento de dados e regras de negócio da aplicação GX2.
A API é responsável por processar requisições, manipular dados e fornecer endpoints para comunicação com o frontend.

📌 Funcionalidades

🔗 API para comunicação com o frontend

📂 Gerenciamento de dados da aplicação

⚙️ Estrutura organizada para rotas e controllers

📡 Manipulação de requisições HTTP

🔒 Base preparada para autenticação e segurança

🛠️ Tecnologias Utilizadas

Node.js

Express

JavaScript

REST API

▶️ Como Executar o Projeto
1️⃣ Clonar o repositório
git clone https://github.com/DiegoScrepante/GX2_Backend.git
2️⃣ Entrar na pasta do projeto
cd GX2_Backend
3️⃣ Instalar as dependências
npm install
4️⃣ Executar o servidor
npm start

O servidor iniciará normalmente em:

http://localhost:3000
📡 Exemplo de Endpoint
GET /api

Resposta esperada:

{
  "status": "API funcionando"
}
